# A1: A Fully Transparent Open-Source, Adaptive and Efficient Truncated Vision-Language-Action Model

> **한 줄 요약**: VLM 백본과 flow-matching action head를 "layer별 warm-start denoising + action-consistency 조기 종료"로 동시에 가속화하여, LIBERO 96.6%와 RoboChallenge 29.0%를 유지하면서 에피소드당 latency를 37.8s → 10.5s로 줄인 완전 오픈소스 VLA.

---

## 1. 배경 및 동기

- VLA는 수십억 파라미터 VLM과 10-20 step diffusion/flow action head에 의존하여 commodity hardware에서 실시간 제어가 어렵다.
- 기존 가속화(양자화, sparsity, early-exit)는 대부분 VLM에만 초점; action head가 새로운 병목이 된다.
- 저자들의 3가지 관찰(Sec. 1): (i) flow-matching trajectory가 3 step 이내에 올바른 모드로 수렴, (ii) 연속 제어 step 간 action redundancy, (iii) 중간 layer의 hidden state만으로도 action 예측에 충분.
- 따라서 "action이 바뀔 때만 compute를 쓰자"는 원칙으로 VLM과 action head를 **동시에** 가속한다.

---

## 2. 방법론

### Action-Consistency Early-Exit (Sec. 3.3.1)
각 layer i에서 예측한 action chunk A_t^(i)와 이전 layer 결과의 불일치 Δ_t^i = d(A_t^(i), A_t^(i-1))가 layer-specific threshold η_i보다 작으면 forward pass 종료. η_i는 training set 기반 layerwise 불일치 분포의 quantile로 offline 보정(Eq. 6-9). 지수 분포 exit_criterion c로 계산 예산을 분포로 변환.

### Inter-Layer Truncated Flow Matching (Sec. 3.3.2)
FM action head가 layer마다 δ=10 denoising을 수행하면 오히려 비용이 올라간다. 해결책: **layer당 δ=2 step만 수행**하고, layer i+1의 초기값을 random noise가 아닌 layer i의 출력(A_t^1(i))으로 warm-start(Eq. 10). 지연 시간이 27.5s → 10.5s로 감소.

### Architecture (Sec. 3.1-3.2)
- Backbone: Molmo (28-layer Qwen2-7B + SigLIP/CLIP vision encoder)
- Action head: Qwen3 ~400M FM expert (KV-conditioned self-attention) OR MLP head (L1 loss)
- Training: layer index i ~ U(0, L) 무작위 샘플링으로 모든 exit point 동시 supervise
- Loss: FM은 conditional flow matching loss(Eq. 1), MLP는 L1
- Pretraining 데이터: DROID, AgiBot, RoboCOIN, RoboMind, GM-100, RoboChallenge + 자체 15,951 trajectory

## 3. 실험 결과

### Simulation (Table 1)
- **LIBERO**: Spatial 97.4 / Object 99.8 / Goal 97.6 / Long 91.4 / **Avg 96.6**. π0.5(96.9), OpenVLA-OFT(97.1)와 동급.
- **VLABench**: Toy 62 / Fruit 64 / Painting 70 / Mahjong 18 / Avg **53.5** (π0.5 49.5 대비 +4).

### Real-world (Table 2)
4개 플랫폼(Franka, AgiBot, OpenArm, Dobot-Arm) 7개 task에서 **mean 56.7%**로 π0(40.8), π0.5(47.5) 대비 각각 +15.9, +9.2. "pick glue" 80%, "fruits (small)" 50% (50-shot).

### RoboChallenge Table30 (Table 4)
Avg **29.00%** — π0(28.33), X-VLA(21.33), RDT-1B(15.00) 능가. Open Drawer 100%, Put Cup on Coaster 90%, Stack Bowls 80%.

### Efficiency Ablation (Tables 3, 5, 6)
- c=1.0에서 computation 15.6% 감소, success 96.6% 유지. c=0.1이면 76.6% 감소하고도 92.3%.
- Inter-Layer Truncated FM이 per-episode time 40.9s → 10.5s, warm-start가 성공률 95.4% → 96.4%.

## 4. 한계 및 미해결 문제

1. **η_i의 data-dependent calibration**: training set에 맞춰 threshold를 fitting하므로 distribution shift가 크면 exit 비율이 바뀔 수 있다. 저자들은 LIBERO-Plus에서도 75.3%로 잘 일반화된다고 하지만, η 재보정 필요성은 남아있다.
2. **VLM-specific**: Molmo 기반으로 초기화된 affordance prior에 의존하는 설계이므로, 다른 VLM backbone으로의 이식성은 명확히 평가되지 않았다.
3. **Action head 종류별 편향**: c=1.0에서 A1-FM이 A1-MLP보다 layer 간 action similarity가 높다는 관찰은 FM이 early exit에 더 적합하다는 것을 의미하지만, 어떤 task/설정에서 MLP 대신 FM을 써야 하는지 가이드가 약하다.
4. Mahjong(VLABench) 18%는 multi-step reasoning task에서 affordance prior만으로 부족함을 드러낸다.

## 5. 총평

| 항목 | 평가 |
|------|------|
| **Novelty** | ★★★★☆ — VLM과 FM head를 **동시에** 가속하고 warm-start로 layer 간 denoising을 이어붙이는 공학적 통합이 참신함 |
| **Practical impact** | ★★★★★ — fully open-source 약속 + 실제로 latency가 4x 감소해 commodity hardware 배포 가능성 매우 높음 |

A1은 새로운 기법을 제안하기보다 "이미 있는 early-exit + flow matching을 어떻게 효율적으로 결합할 것인가"라는 엔지니어링 문제에 깨끗한 답을 제시한다. Warm-start denoising은 단순하지만 inference time을 4배 줄이는 결정적 아이디어이며, 29.0% RoboChallenge 결과는 closed-source 시스템에 견줄 만한 완전 오픈 VLA가 가능함을 보여준다.
