# E₀: Enhancing Generalization and Fine-Grained Control in VLA Models via Continuized Discrete Diffusion

> **한 줄 요약**: Discrete action token에 Gaussian noise를 적용하는 "continuized discrete diffusion"으로, 2048 bins까지 확장 가능하며 14개 환경(LIBERO+VLABench+ManiSkill)에서 π₀/π₀.₅ 대비 **평균 +10.7%** 향상, real-world 45.6%, 단일 RTX PRO 6000에서 24시간 학습.

---

## 1. 배경 및 동기

### 기존 연구의 구조적 한계
- **Autoregressive VLA** (OpenVLA): 256-bin 한계, fine-grained control 부족
- **Continuous diffusion**: VLM의 discrete token 공간과 mismatch → "off-support prediction" (discrete action set 밖의 값)
- **Discrete diffusion** (DD-VLA): Binary mask만 가능, noise granularity 제한

### 핵심 질문
- **Discrete token에 continuous Gaussian noise를 적용하면, 256-bin 한계를 넘어 2048+ bin까지 확장 가능한가?**

---

## 2. 방법론 심층 분석

### 2.1 Continuized Discrete Diffusion

One-hot action token에 smoothing (α=0.1)을 적용한 후 Gaussian noise:

$$\tilde{A}_t^\tau = \tau \tilde{A}_t + (1-\tau)\epsilon, \quad \epsilon \sim \mathcal{N}(0, I)$$

- τ: Beta분포에서 샘플 (higher noise에 편향)
- 학습 목적함수: **Cross-entropy loss** (VLM과 동일)

### 2.2 Architecture

- **VLM**: PaliGemma (Gemma, width=2048, depth=18, heads=8)
- **Action expert**: **300M parameters** (width=1024, depth=18)
- **Action discretization**: **최대 2048 bins** (1st-Nth percentile quantile)
- **Action chunking**: H=50 timesteps
- **Action dim**: D_a=7 (EE) or D_a=8 (joint)

### 2.3 이론적 주장

**Lemma 1**: Discrete actions → VC dimension O(K log K). Continuous → unbounded pseudo-dimension. → "Tighter sample-complexity guarantees"

**Lemma 2**: Continuous diffusion → off-support averaging (discrete set 밖 값 생성). Discrete diffusion → "preserves action support with probability 1"

> ❓ **예상 질문**: Off-support prediction이 실제로 문제가 되는가?
> **답변**: 이론적으로는 문제. 실험적으로는 continuous diffusion도 잘 작동 (CogACT, pi0). Off-support 값이 nearest valid action에 round되면 실질적 차이가 작을 수 있음. E₀의 이론적 주장은 실험적 관찰보다 강함.

## 5. Ablation 분석

### Discretization Bins

| Bins | LIBERO Avg (%) |
|------|---------------|
| 128 | 0.0 (failure) |
| 256 | 75.55 |
| 512 | 79.95 |
| 1024 | 81.65 |
| **2048** | **84.30 (optimal)** |
| 4096 | 83.55 |
| 8192 | 82.00 |

- **2048이 sweet spot**. 128은 완전 실패, 256 대비 +9%p

## 6. 한계 및 미해결 문제

1. **Real-world 개선 modest**: 45.6% vs π₀ 43.1% = +2.5%p. Sim과 달리 real에서 차이 작음
2. **이론 vs 실험 괴리**: Off-support 문제의 이론적 중요성에 비해 실험적 차이가 modest한 경우 있음 (LIBERO-Goal: π₀.₅가 E₀ 능가)
3. **128-bin failure**: 왜 128에서 완전 실패하는지 분석 부족
4. **Action chunking H=50에서의 하락**: 49.65%로 급락 — long horizon prediction의 한계
